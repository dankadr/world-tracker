Where should the Admin Panel live?

How you access the admin panel in the app

A

Hidden in Settings

A button appears at the bottom of the Settings panel — only visible when logged in as dankadr100@gmail.com. No visible trace for other users.

  • No nav clutter
  • Stealth — users never see it
  • Two taps to reach (Profile → Settings → Admin)
B

Admin Tab (mobile) / Sidebar item (desktop)

A 5th tab appears in the bottom tab bar on mobile and as a nav item in the sidebar on desktop — only rendered when you're logged in.

  • One tap away
  • Feels like a proper section
  • Slightly odd to have a 5th tab only sometimes visible
C

Profile screen shortcut

A discreet "Admin" button appears directly on your profile screen (not buried in Settings) — visible only to you.

  • Prominent but still personal
  • One tap from Profile tab
  • Profile screen gets busier

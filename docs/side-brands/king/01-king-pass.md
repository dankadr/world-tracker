# King Plan 01 — King Pass

**Goal:** Package premium features into a more desirable, identity-driven membership instead of a plain subscription toggle.

## Ideas

- Replace generic “Premium” language with a **King Pass** concept
- Bundle unlimited bucket list items, richer exports, collab maps, and advanced photo/media tools
- Add status cosmetics: profile frames, rare map themes, prestige badges
- Include “seasonal drops” of limited-time challenges or themes

## MVP

- New paywall copy and visual treatment
- King badge on profile and share cards
- Premium feature matrix tuned around identity + status
- Analytics on conversion by feature and theme

## Follow-ups

- Giftable passes
- Annual “Founding King” tier
- Referral unlocks

# King Plan 04 — King Concierge

**Goal:** Introduce a premium planning / recommendation layer that feels like a travel strategist.

## Feature Ideas

- Bucket-list-to-itinerary suggestions
- “Best next country” recommendations based on the user’s gaps and past travel
- Regional completion suggestions (e.g. “you are 3 states away from finishing New England”)
- Seasonal destination inspiration
- Smart prompts for trip timing, packing, and route order

## MVP

- Read-only recommendation cards inside bucket list and profile
- No booking integrations required initially
- Use existing tracker + wishlist data before adding external travel APIs

## Longer Term

- AI-generated day plans
- Budget-aware route suggestions
- Concierge share threads for couples / groups

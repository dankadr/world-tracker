# King Plan 02 — King Quests

**Goal:** Add curated, high-status challenge ladders that feel more meaningful than generic achievements.

## Example Quest Lines

- **Capital King** — visit 10 / 25 / 50 capitals
- **Island King** — track island-heavy travel goals
- **Rail King** — complete iconic rail corridors
- **World Wonder King** — UNESCO + landmark bundle quests
- **Continental King** — finish milestone sets across every continent

## Product Shape

- Multi-step quest arcs
- Better progress storytelling than flat badge unlocks
- Reward cards with rarity / completion %
- Optional friend leaderboard per quest line

## Dependencies

- Better achievement metadata
- Public profiles / social feed work
- Potential premium gating for advanced quest sets

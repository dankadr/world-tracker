# King — Side Brand Concept

**Created:** 2026-03-19
**Status:** New idea bank
**Positioning:** A premium-adjacent side brand for bolder, more aspirational travel experiences inside the World Tracker ecosystem.

## Concept

**King** is the luxe / challenge / bragging-rights side brand. Instead of focusing on basic trip logging, it packages the most ambitious and socially shareable features under a more editorial, high-status identity.

Think:
- prestige travel stats
- elite-style challenge ladders
- beautifully shareable travel recaps
- premium-feeling exploration tools

## Brand Pillars

1. **Prestige** — make progress feel collectible and worth showing off
2. **Storytelling** — turn travel data into rich narratives, not just percentages
3. **Competition** — add tasteful, opt-in rivalry and rankings
4. **Curation** — highlight iconic routes, rare achievements, and destination sets

## Planned Feature Tracks

- `01-king-pass.md` — membership / perks / premium packaging
- `02-king-quests.md` — aspirational challenge ladders and curated travel missions
- `03-king-postcards.md` — luxury share cards and social storytelling outputs
- `04-king-concierge.md` — AI itinerary, inspiration, and destination guidance
- `05-king-clubs.md` — invite-only groups, clubs, and status tiers

## Notes

This side brand is intentionally strategy-heavy first. It can start as a themed in-app area, then evolve into a pricing tier, seasonal campaign, or standalone landing page if it resonates.
